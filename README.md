## セット手順

python+pip+django

```
pip install pipenv # pipでpipenvを入れる
pipenv --three # python3セット
pipenv install # pipfileに記述されたものがインストールされる
pipenv shell # pipenvのshellモード
cd dp
python3 manage.py runserver # djangoが起動
exit # pipenv shellモードから抜ける
```

npm

```
cd dp
npm install
yarn
```
npmまたはyarnを使用
